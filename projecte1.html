PROJECT 1: Static Routing -ACL Packet Tracer

1. Introduction & Objectives

In this project, we implemented routing between routers on different subnets to create a central network for a company in Cisco Packet Tracer. Each subnet contains connected PCs, laptops, and servers, ensuring connectivity between them all. We also configured Access Control Lists (ACLs) to maintain a secure system and control over all data shared between subnets.

2. Technical Process

"The first step was to set the network card to monitor mode capture all traffic:"

terminal@kali:~$
sudo airmon-ng start wlan0
sudo airodump-ng wlan0mon

"Next, the Handshake was captured to verify password strength. The results demonstrated that using common dictionaries allows access in less than 5 minutes."

3. Conclusions

"It is critical to implement robust passwords and, if possible, use digital certificates (WPA2-Enterprise) instead of pre-shared keys (PSK) in corporate environments."
